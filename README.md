# LensLib

📱 **Overview**
LensLib is a photography app that allows users to browse, upload, and manage HD wallpapers. The app provides smooth browsing and a visually appealing gallery experience.

🎯 **Problem it Solves**
Provides users with a platform to discover, upload, and manage high-quality wallpapers easily.

✨ **Features**
- Splash screen, login/register
- Browse gallery of HD images
- Upload & manage images
- User profile & settings
- Smooth UI/UX with dark mode support

🛠 **Tech Stack**
- Flutter
- Firebase Authentication & Firestore
- Image caching libraries

🧠 **Architecture**
- MVVM / Clean Architecture
- Layered separation of UI, services, and data
